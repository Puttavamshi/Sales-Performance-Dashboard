# Sales-Performance-Dashboard
📊 Sales Performance Dashboard – Power BI
📝 Project Overview
This project presents an interactive Sales Performance Dashboard built using Power BI to analyze business performance across regions, customer segments, and product categories. The dashboard provides key insights to support data-driven decision-making.

🎯 Objectives
Monitor overall sales performance using key KPIs
Analyze customer segments and regional trends
Identify top-performing products and categories
Understand seasonal sales patterns
📊 Key Features
KPI tracking: Total Sales, Total Orders, Total Quantity, Average Order Value
Region-wise and category-wise sales analysis
Customer segmentation insights
Monthly sales trend visualization
Top-performing products and customers
Interactive filters for dynamic analysis
💡 Key Insights
Consumer segment contributes the highest revenue
Phones and Chairs are top-performing sub-categories
West region leads overall sales performance
Sales show strong growth during Q4 (Nov–Dec)
🛠️ Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Power Query
Excel
📁 Dataset

The dataset includes sales transaction data with details such as order date, product category, region, customer segment, and sales metrics.

🚀 Outcome

This dashboard enables quick identification of business trends and supports decision-making by providing clear, interactive, and insightful visualizations.

📌 How to Use
Open the Power BI (.pbix) file
Use slicers (Year, Category, Region) to filter data
Explore visuals to analyze trends and insights
